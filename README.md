esoTalk是一个拥有众多优势的免费，开源的基于PHP和MySQL下的网页应用。
她的设计极为简介，速度极快，超轻量级别并且有良好的可拓展性。  

###特征
1.没有版块的分区，几乎都是tag实现的.  
2.因为tag实现，所以search功能十分强大.  
3.及时性很强，完全可以像IM一样随时聊天喔.  
4.插件很强大，很好用.  
5.程序很精炼，不到700K的大小！  
6.本版本为官方1.0.0.3简体中文汉化版. (http://esotalk.net).  

### 安装简体中文版 esoTalk  
1.上传包内文件到网站根目录.  
2.访问网站首页.  
3.安导向导会指导你安装程序.   
4.中文交流论坛(http://esotalk.net).
